### 입력 명령어: ***GWi"[END]"[ESC]ZZ*** 
> **G**: 마지막 줄로 이동, **W**: 다음 단어 앞으로 커서 이동\
> **i**: INSERT MODE,  **[END]**: 라인 맨 끝으로 이동\
> **[ESC]**: INSERT MODE 종료, **ZZ**: 저장 후 종료
